Built Jarvis in NodeJS and it connects as user to GTalk. So I'm talking to Jarvis 24/7.

What does Jarvis do ?

Checks the News


Google is stopping reader so I was looking for new one. But never found one I liked and built it into Jarvis.


So every 30 min Jarvis checks all my registered rss / news feeds.


Then has a trained Bayesian Classifier on what news I like. So I only get my news, mostly about tech and not ever anything about Barbie that have a new model ...

Yoda doesn't read the news too well

Downloads my Youtube Subscriptions


Runs every morning 11AM and downloads all the latest videos from my Youtube Subscriptions.


And tells me during the day when and what videos are downloaded.


Also accepts url to download with the command youtube {url} or youtube mp3 {url} to download the mp3.

Tells me about public holidays


Tells me the day before and on the day if today is a public holiday.


Scraping this from SA Goverment site


Because my memory is just that bad !

Checks for me on the Internet


Checks the Google Alerts feed every hour to see if my name pops up anywhere online.


Some advance stuff here.

Downloads my Podcasts


Every Morning at 4AM my podcasts are downloaded and I'm notified as soon as they are done.


The podcasts for all my shows !

Weather


Send me the weather forcast for the day every morning at 6AM.


Also keeps this as record and runs it through the prediction API from Google to see if I can do better based on averages and past conditions. Both results are sent.

Checks DSTV


Has a list of shows I like loadded into a Bayesian Classifier and tells me when shows that I would like are showing tonight.


Does the same with Movies.

And only does this in certain hours




Weekdays only from 6AM to 10PM

Holidays only from 10AM to Midnight

Weekends only from 11AM to 3AM (THE NEXT MORNING)

Because App Engine

App Engine makes it so easy to build a chat bot


import com.google.appengine.api.xmpp.JID;
import com.google.appengine.api.xmpp.Message;
import com.google.appengine.api.xmpp.MessageBuilder;
import com.google.appengine.api.xmpp.SendResponse;
import com.google.appengine.api.xmpp.XMPPService;
import com.google.appengine.api.xmpp.XMPPServiceFactory;

JID jid = new JID("example@gmail.com");
String msgBody = "Hello Master !";
Message msg = new MessageBuilder()
    .withRecipientJids(jid)
    .withBody(msgBody)
    .build();

boolean messageSent = false;
XMPPService xmpp = XMPPServiceFactory.getXMPPService();
SendResponse status = xmpp.sendMessage(msg);
messageSent = (status.getStatusMap().get(jid) == SendResponse.Status.SUCCESS);

if (!messageSent) {
    // Send an email message instead...
}
