Part #1 - Intro to NodeJS

Part #1 - Bit of Background

NodeJS was created by Ryan Dahl and has it's development sponsored by Joyent, Ryan's former Employer.

His orgiganal goal was to create a simple to use framework that could be used to create apps with push capiblities like GMail

After trying a few languages he chose Javascript due to it's lack of an existing IO API he could define a Non-Blocking I/O API.

And he ended up with NodeJS

Part #1 - NodeJS V8

NodeJS is easiest described standalone V8 with a IO wrapper to support writing to files and networking.

This allows you to write server-side applications in javascript. Running in the incredibly fast V8 engine.

Part #1 - Single Threaded

As NodeJS is using V8. The entire NodeJS process is single threaded. All your functions etc. are executed in this thread.

With API like cluster NodeJS can share and load balance accross a port with as many threads running as you wish.

Part #1 - NPM

What framework is complete without a package manager ?

Node's package manager is NPM. Which started as a seperate library but was included by default as of V0.6.

Uses the NPM registery https://npmjs.org/ to download modules and adds them to your node_modules folder
